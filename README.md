# 🏙️ Chongqing 24/7 Autonomous AI Agency

Sistem AI Agent mandiri dengan estetika Cyberpunk Chongqing (China) yang dirancang untuk menghasilkan konten, copywriting, strategi pemasaran secara otonom 24 jam non-stop menggunakan model AI lokal (tanpa bergantung pada API pihak ketiga seperti OpenAI).

## 🚀 Fitur Utama
- **100% Custom AI Engine:** Menggunakan model AI lokal (Ollama / Llama 3) dengan API Key buatan sendiri.
- **Cyberpunk Chongqing UI:** Tampilan antarmuka web futuristik dengan animasi partikel neon bergerak 24/7.
- **Autonomi Penuh:** Siap memproses tugas konten dan pemasaran secara mandiri.
- **Integrasi Keuangan:** Simulasi *webhook* pembayaran otomatis terhubung ke akun DANA.
- **Sistem Pengaman / Notifikasi:** Siaga mengirimkan *alert* kendala operasional langsung ke WhatsApp Anda.
